# 🚀 Task Management System

A modern Full Stack Task Management Application built using React, Node.js, Express, and MongoDB.  
This application helps users efficiently manage daily tasks with authentication, task tracking, and responsive UI.

---

## 📌 Features

- 🔐 User Authentication (Login & Register)
- ✅ Create, Update, Delete Tasks
- 📋 Task Status Management
- 🌙 Responsive Modern UI
- ⚡ Real-time User Experience
- 🔒 Secure Backend APIs
- 📱 Mobile Friendly Design

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- CSS / Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
